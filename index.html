KraveAI - Fotos Reales de Instagram

Solución para mostrar avatares reales de usuarios de Instagram

¿Cómo funciona?

  • El sistema utiliza unavatar.io para obtener las fotos reales de Instagram
  • Si la imagen no está disponible, se muestra un avatar con las iniciales del usuario
  • Las imágenes se cargan automáticamente al agregar un usuario
  • Para mejores resultados, asegúrate de que el perfil de Instagram sea público
